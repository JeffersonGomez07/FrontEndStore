Frontend Store
Descripción

Frontend Store es una tienda online moderna desarrollada únicamente con HTML y CSS, completamente responsive y optimizada para todos los dispositivos. Ofrece una experiencia de usuario fluida y navegación intuitiva, enfocada en mostrar productos de manera atractiva y funcional.

Características

Diseño responsive adaptable a móviles, tablets y escritorio.

Interfaz limpia y profesional.

Experiencia de usuario optimizada.

Uso exclusivo de tecnologías de frontend: HTML y CSS.

Estructura organizada para fácil mantenimiento y escalabilidad futura.

Estructura del Proyecto
frontend-store/
│
├── index.html           # Página principal
├── productos.html       # Página de productos
├── contacto.html        # Formulario de contacto
├── css/
│   ├── normalize.css    # Reset de estilos
│   └── style.css        # Estilos personalizados
├── js/
│   └── main.js          # Funcionalidades opcionales del frontend
└── img/
    ├── productos/       # Imágenes de productos
    └── logo.png         # Logotipo de la tienda
Tecnologías

HTML5

CSS3

Diseño responsive

Cómo ejecutar el proyecto

Clonar el repositorio:

git clone https://github.com/tu-usuario/frontend-store.git

Abrir index.html en un navegador moderno (Chrome, Firefox, Edge, Safari).

Navegar por las páginas de productos y contacto.
